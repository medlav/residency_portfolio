PortfolioEntry

PortfolioEntry class

Represents an individual clinical or educational activity recorded in the residency portfolio.

This class encapsulates all details regarding a specific entry, including its duration, the quantity of the activity performed (amount), the medical institution, and the specific PortfolioCategory it satisfies.
