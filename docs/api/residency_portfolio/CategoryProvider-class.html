CategoryProvider

CategoryProvider class

A state management provider that handles the lifecycle of medical portfolio categories.

This provider manages the loading, persistence, and modification of PortfolioCategory objects using SharedPreferences. It allows for specialty-specific defaults and custom user-defined categories.
